## Stock Forecasting and Anomaly Detection using Hadoop and Apache Spark

### Overview

This project presents a scalable data analytics pipeline for stock price forecasting and anomaly detection using distributed computing technologies. The system integrates Hadoop Distributed File System (HDFS) for storage and Apache Spark for parallel data processing and machine learning.

The objective is to predict next-day stock prices based on historical data and identify anomalous market behavior using statistical methods.
